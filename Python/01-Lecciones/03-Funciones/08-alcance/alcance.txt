Tema 26: Alcance de las Variables en Python:

El alcance en Python determina en quﾃｩ partes del cﾃｳdigo se puede acceder a una variable. Hay variables locales y globales, 
y dependiendo de dﾃｳnde se declaren, podrﾃ｡n ser utilizadas en distintos lugares del cﾃｳdigo.


Tipos de Alcance en Python:

ﾃ［bito Local (Local Scope)
    Una variable declarada dentro de una funciﾃｳn solo existe dentro de esa funciﾃｳn.
    No se puede acceder a ella desde fuera de la funciﾃｳn.

ﾃ［bito Global (Global Scope)
    Una variable declarada fuera de cualquier funciﾃｳn puede ser usada en todo el cﾃｳdigo.
    Sin embargo, dentro de una funciﾃｳn, no puedes modificar una variable global directamente, a menos que uses la palabra clave global.


Ejemplos Explicados:

-----------------------------------------------------------------------------------------------------------------
Ejemplo 1: Variable de ﾃ［bito Local

笨 Explicaciﾃｳn:

-----------------------------------------------------------------------------------------------------------------
Ejemplo 2: Variable de ﾃ［bito Global

def mostrar_mensaje():
    print(mensaje)  # Se puede acceder a la variable global

mostrar_mensaje()
print(mensaje)  # Tambiﾃｩn se puede acceder fuera de la funciﾃｳn

笨 Explicaciﾃｳn:
Aquﾃｭ, mensaje se declarﾃｳ fuera de la funciﾃｳn, por lo que puede ser accedida en cualquier parte del cﾃｳdigo.

Ejemplo 3: Modificar una Variable Global Dentro de una Funciﾃｳn (global)

    global contador  # Indicamos que vamos a modificar la variable global
    contador += 1
    print(f"Contador dentro de la funciﾃｳn: {contador}")

笨 Explicaciﾃｳn:

Sin global, Python tratarﾃｭa de crear una variable nueva dentro de la funciﾃｳn en lugar de modificar contador.
Usamos global contador para indicar que queremos modificar la variable global.

沒 Resumen
Las variables locales existen solo dentro de una funciﾃｳn.
Las variables globales pueden ser accedidas en cualquier parte del cﾃｳdigo.
Para modificar una variable global dentro de una funciﾃｳn, debes usar global.
Si intentas modificar una variable global sin global, Python crearﾃ｡ una nueva variable local en su lugar.
